Apresentação do Projeto "TechnoTrends": Uma Plataforma de Venda de Periféricos Eletrônicos

A página busca oferecer uma experiência única aos usuários, disponibilizando uma variedade de produtos de alta qualidade que aprimoram a experiência tecnológica cotidiana.

Com as tecnologias fundamentais: HTML, CSS e JavaScript. Ao longo desta apresentação, exploraremos as diferentes etapas dessa página.

Etapa 1: Página de Login

A primeira etapa é a página de login. Através da linguagem HTML, construí um formulário simples, permitindo aos usuários inserirem suas informações de acesso. Isso é estilizado com a linguagem CSS, resultando em um design amigável, com cores coerentes.

Etapa 2: Página Inicial com Produtos

Ao acessar a página principal, os usuários são recebidos por uma exibição de produtos eletrônicos. Utilizando HTML e CSS, organizei os produtos em uma grade, tornando fácil para os visitantes explorarem diferentes categorias, como notebooks, computadores, hardwares, monitores, periféricos e games. Cada categoria é representada por uma imagem.

Funcionalidade de Busca Dinâmica com JavaScript: Aqui, o JavaScript entra em ação. Com a implementação da função searchProduct(), os usuários podem inserir palavras-chave na barra de pesquisa e obter resultados em tempo real. A função é acionada ao clicar no botão "Buscar", melhorando significativamente a usabilidade do site e auxiliando os usuários a encontrar produtos específicos.

Etapa 3: Página de Propósito do Site

Na página que explora o propósito do site, destaco a relevância do "TechnoTrends" na oferta de soluções tecnológicas. Usando HTML e CSS, criamos um layout simples e legível, apresentando informações sobre a missão da página.

Barra de Navegação e Rodapé Tanto na página inicial quanto na de propósito, a barra de navegação superior desempenha um papel crucial, permitindo o acesso rápido a diferentes seções, como suporte, carrinho de compras e área do usuário. O rodapé, por sua vez, fornece links para termos de uso, política de privacidade, contato e informações adicionais sobre o projeto, tudo construído com HTML e estilizado com CSS.
